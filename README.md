# cmdi

Cmdi (command-ee) is a console system for Roblox, taking inspiration from projects like [Conch](https://github.com/alicesaidhi/conch), [Cmdr](https://github.com/evaera/Cmdr), and [Centurion](https://github.com/paradoxuum/centurion).

This repository consists of multiple packages, located within the [`packages/`](packages/) directory.

- [`cmdi_language`](packages/cmdi_language/)\
  The language behind Cmdi. Similar to other scripting languages, like Powershell.
